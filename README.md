## ⚡️ Quick start

> [!IMPORTANT]
> Don't forget adding OPENAI_API_KEY

1. clone the repo
   ```console
   git clone https://github.com/Gion-AI/ezil.git
   ```
2. install packages
   ```console
   pnpm install
   ```
3. start client
   ```console
   node ezil
   ```
